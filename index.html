Partições

    sda, significa SATA disco
    sda1, partição primária
    sda2, partição primária
    sda5, partição lógica

    hda, ATA disco

    Partições primárias, máximo 4 partições
    Partições lógicas, abre mão de uma lógica e abre extendidas, limite 12

    sdc6, terceiro disco, segunda partição lógica
    sdd2, quarto disco, segunda partição primária
    sdb9, segundo disco, quinta partição lógica

    b - Dispositivo de Bloco, armazena informação
    c - Dispositivo de Caracter, recebe e envia informação
    d - Diretório
    l - Link simbólico
    tty1, significa terminal, CTRL + ALT + F1
    tty2, significa terminal, CTRL + ALT + F2
    tty3, significa terminal, CTRL + ALT + F3
    tty4, significa terminal, CTRL + ALT + F4
    tty5, significa terminal, CTRL + ALT + F5
    lp, significa porta de impressora, LPT1
    ttys1, significa porta de comunicação, COM1
    psaux, significa mouse
    net, significa rede
